## 💎 Elite Features

### 1. 🌍 End-to-End AI Video Dubbing (NEW!)
Break language barriers instantly. Our fully automated pipeline takes any video and dubs it into **Hindi, Marathi, Spanish, French, or English**.
- **Audio Extraction**: `MoviePy` strictly extracts the original audio track.
- **Speech-to-Text**: `OpenAI Whisper` transcribes the exact spoken words with millisecond timestamp precision.
- **Translation**: `HuggingFace MarianMT` & Machine Translation networks map the meaning into the target language.
- **Neural TTS Generation**: `Edge-TTS` generates ultra-realistic, studio-quality synthetic voice acting.
- **Auto-Merging**: The new language track is perfectly merged back into the video for a seamless, ready-to-download `.mp4`!

## 🚀 Quick Start Instructions

### 1. Install Dependencies
```bash
pip install openai-whisper transformers torch edge-tts googletrans==4.0.0-rc1 qiskit-aer qiskit fastapi uvicorn moviepy opencv-python librosa yt-dlp
```

### 2. Launch Background Server
```bash
cd backend
python -m uvicorn main:app --port 8000 --reload
```

### 3. Launch Frontend Dashboard
```bash
cd frontend
npm run dev -- --port 5174
```
